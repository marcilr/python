Interact with SQL database using only Python 2.4
There is a Python MySQL Module, called MySQLDB,
which supports Python 2.3-2.7:
  http://sourceforge.net/projects/mysql-python/
User Guide can be found here:
  http://mysql-python.sourceforge.net/MySQLdb.html
Check out pysqlite. It is the same module as in the
newer python versions and I have used it no prob in
2.4 before.
  http://trac.edgewall.org/wiki/PySqlite
