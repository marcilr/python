MySQLdb is an thread-compatible interface to the popular MySQL database
server that provides the Python database API.[1]
--mysql-python.sourceforge.net/MySQLdb.html

MySQLdb is an interface for connecting to a MySQL database server from
Python.  It implements the Python Database API v2.0 and is built on top
of the MySQL C API.
--www.tutorialspoint.com/python/python_database_access.htm
